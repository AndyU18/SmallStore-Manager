# Carpeta de Componentes Comunes

Aquí van los componentes de interfaz reutilizables:
- `/ui`: Botones, Inputs, Modales, Alertas genéricas.
- `/layout`: Sidebar, Navbar, contenedores de páginas.
- `/tables`: Componentes basados en TanStack Table para mostrar listas.
- `/charts`: Gráficas basadas en Recharts para reportes y dashboard.
