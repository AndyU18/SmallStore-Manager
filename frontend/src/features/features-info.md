# Carpeta de Features (Funcionalidades)

Organización por dominios de negocio:
- `/products`: Formularios de producto, filtros, modals específicos de producto.
- `/sales`: Carrito de ventas, buscador de productos rápido, modal de pago.
- `/categories`: Modales de creación/edición de categorías.
- `/stock`: Formularios de ajuste de stock y auditoría.
- `/dashboard`: Tarjetas de KPI y gráficos integrados del dashboard.
